# 신재생에너지 과제

> 지원동기

*신재생에너지 산업은 단순한 발전기술을 넘어, 시스템 설계·데이터 분석·운영 최적화가 융합된 복합 산업으로 발전하고 있습니다.
저는 친환경·에너지 컨설팅 경험을 바탕으로 기술과 데이터를 연결해 새로운 가치를 만드는 ‘에너지 기획자’로 성장하고자 합니다.
이번 교육과정을 통해 신재생에너지와 IoT 기술의 실무 감각을 익히고, 나아가 미래 에너지 전환을 선도할 프로젝트를 기획할 수 있는 역량을 기르고 싶습니다.*
